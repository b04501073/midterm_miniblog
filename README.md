# 題目名稱
- Mini Blog

# 簡介
- 一個具有發文、搜尋功能的簡易部落格

# 操作方式
## build
- npm install
- node server.js
- npm start
## run
- 點選New Post
- 輸入標題與內容並送出
- 返回Home頁面查看所有貼文或到Search頁面輸入關鍵字查詢相關貼文

# 使用的框架
### 前端
- React
- React-router-dom
### 後端
- Mongoose
- Socket
