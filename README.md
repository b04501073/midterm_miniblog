# 題目名稱
- Mini Blog

# 簡介
- 一個具有發文功能的簡易部落格

# 操作方式

### App
- 點選NEW POST
- 輸入標題、內容
- 送出 並回到HOME查看
### Build
- npm install
- node server.js
- npm start

# 使用的框架
### 前端
- React
- React-router-dom
### 後端
- Mongoose : 連接mongodb資料庫
- Socket.io: 串連前端與後端

# 我的貢獻
前端架構、設計都是自己來，後端主要參考老師的範例，並且稍做修改
